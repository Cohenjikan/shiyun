# 诗云 · 2026 年 6 月更新日志

> 一句话:这次更新分两块 —— **界面体验**重做了一批,**诗词数据**修了一批"搜不到 / 名字对不上"的小毛病。

---

## 一、界面 / 体验更新

1. **「留影」做图功能大改**
   把喜欢的诗句做成一张可保存的图片。现在:
   - 左下角一个**齿轮按钮**统一管理所有设置,界面更清爽;
   - 可以**开关背景底色**、**自由挑任意文字颜色**、**隐藏顶部说明文字**;
   - 诗句固定**竖排**,更有古意;
   - 新增**可自由拉伸的取景框**:拖右下角手柄改大小、两指捏合或滚轮缩放、拖动整体挪位、一键复位;框里的字号会**自动调整、刚好填满**,不再留一堆空白。

2. **诗体标注修正**
   以前通过分享链接打开、或定位过去的某些诗(比如七言律诗),"诗体"会被错标成"自由诗"。现在**显示正确了**。

3. **手机 / 平板操作更顺手**
   触屏默认改成**"锁定整体旋转"**:一根手指转动整片星空,两指捏合或滚轮缩放(以前**只能拖、不能缩放**,很别扭)。点某位诗人或某首诗即可切换聚焦对象。电脑端保持原来的自由飞行。

4. **可以关掉"随机出诗"**
   新增一个开关:不想要"点空白处就随机蹦出一首诗"时,可以关掉它(默认开着)。

5. **新手引导重做**
   第一次打开时的引导,重写成更清楚的**四步**,并区分电脑 / 手机的操作说明。(老用户会再看到一次。)

---

## 二、诗词数据修正

1. **皇帝按"年号 / 庙号"也能搜到了**(新增 21 位)
   以前只能用本名搜。现在搜这些称呼也能找到对应诗人,例如:
   乾隆、康熙、雍正、汉武帝、隋炀帝、唐太宗、唐玄宗(唐明皇)、武则天、宋徽宗、明太祖、海陵王…… 等共 21 位。

2. **生僻字名字搜得到了:王禹偁**
   北宋诗人"王禹偁"的"偁"是个生僻字,库里当初用了同音的"王禹称",导致按正确写法搜不到。现在搜"**王禹偁**"也能找到了(收 729 首)。

3. **最常用的名字搜得到了:蔡文姬**
   以前搜"蔡文姬"找不到(库里记的是她的本名"蔡琰")。现在搜"**蔡文姬**"能直接找到。

4. **名家名字统一,修好两个"点了没反应"的条目**
   "陶渊明 / 陶潜""蔡文姬 / 蔡琰"这两位,系统内部名字没对齐,导致他们在"名家"里其实是**失效条目**(排序、强调都不生效)。现在统一用库内本名修好了 —— 而用常用名照样能搜到(见上)。

5. **名篇能按标题搜到了**(修"少而精"诗人被埋没)
   有一批"作品不多、但篇篇是名篇"的诗人,以前因为同名标题的高产诗人太多,把他们挤出了搜索结果,导致**按标题搜不到他们的招牌作**。现在把这 14 位提为"优先",修好了。涉及:
   张若虚《春江花月夜》、张继《枫桥夜泊》、崔颢《黄鹤楼》、王之涣 / 王翰《凉州词》、贺知章《回乡偶书》、张志和《渔歌子》、金昌绪《春怨》、王湾《次北固山下》、林升《题临安邸》、叶绍翁《游园不值》、王冕《墨梅》、张养浩《山坡羊》、宋之问 等。

6. **(内部)文档数字更新 + 新增勘误台账**
   把项目文档里的统计数字更新到最新(诗人 32,657、诗 933,857、赠诗关系 4,976);并新增两份内部文档,记录所有已发现的问题与未来补充计划。这两项对使用者无感。
